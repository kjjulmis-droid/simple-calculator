# Simple Calculator Project

## Project Description
A comprehensive console-based calculator application developed in Python. This project demonstrates core programming concepts including Object-Oriented Programming (OOP), exception handling, and modular design.

## Features
- **Basic Arithmetic**: Supports addition, subtraction, multiplication, and division.
- - **Calculation History**: Keeps track of all operations performed during the session.
  - - **Error Handling**: Robust validation for numerical inputs and prevention of division by zero.
    - - **Interactive Menu**: User-friendly console interface for easy navigation.
     
      - ## Python Concepts Demonstrated
      - - **Object-Oriented Programming (OOP)**: Implementation of a `Calculator` class with encapsulated methods and attributes.
        - - **Data Structures**: Use of lists for history tracking and dictionaries for operation mapping.
          - - **Exception Handling**: `try-except` blocks to manage invalid user inputs.
            - - **Control Flow**: `while` loops and conditional statements for program logic.
             
              - ## How to Run
              - 1. Ensure Python 3.x is installed on your system.
                2. 2. Clone this repository or download `main.py`.
                   3. 3. Run the script using:
                      4.    ```bash
                               python main.py
                               ```
